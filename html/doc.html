Documentation

  • javascript version
    • <script type="text/javascript" src="jsonfn.js"></script>
    • No configuration is needed
    • Plugin uses native JSON object, which is implemented in most modern browsers. It's tested in Firefox, Chrome, Opera, IE9. It doesnt work in IE8.
    • Usage:
      • Stringify
        var str = JSONfn.stringify(obj);
            @obj - Object;
            @return - String;
      • Parse
        var obj = JSONfn.parse(str[, date2obj]);
            @str - String;
            @date2obj - Boolean (optional); if true, string in ISO8061 format is converted into a Date object; otherwise, it is left as a String.
            @return - object;
      • Clone
        var obj_clone = JSONfn.clone(obj[, date2obj]);
            @str - String;
            @date2obj - Boolean (optional); set it true if obj has property instanceof Date
            @return - Object;
  • Node.js version:
    • download: npm install jsonfn
    • code: https://github.com/vkiryukhin/jsonfn-node
    • Usage:
      • Stringify
        var strfn = require('jsonfn').JSONfn.stringify(obj);
            @obj - javascript object;
            @return - string;
      • Parse
        var objfn = require('jsonfn').JSONfn.parse(strfn);
            @str - string;
            @return - object;
