Documentation

  • <script type="text/javascript" src="jsonfn.js"></script>
  • No configuration is needed
  • Plugin internally uses JSON object, which is native for modern browsers. For old browsers, which don't support JSON, include Douglas Crockford's json2.js
    https://github.com/douglascrockford/JSON-js )
  • Usage:
    • Stringify

      var str = JSONfn.stringify(obj);
      @obj - javascript object;
      @return - string;

    • Parse

      var obj = JSONfn.parse(str);
      @str - string;
      @return - object;
