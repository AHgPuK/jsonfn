Overview

JSONfn is a javascript plugin to convert jsvascript object with functions into a string and convert this string back to the object, as well as clone such kind of object.

With this plugin you can easily serialize any javascript object, save this string in a local storage or on a server and then get the string and convert it back to object. I repeat it: object with functions!

If you still reading this text, you probably don't need any other arguments to start using this plugin. Well, one more thing: full development version with comments is only 1k (2K with license, author, etc :).

Node.js version is also available (see Documentation)
