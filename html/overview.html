Overview

JSONfn is a plugin to convert jsvascript objects with functions into a string and convert this string back to the object.

With this plugin you can easily serialize any javascript object, save this string in a local storage or on a server and when you need it, get the string and convert it back to the object. The main advantage this plugin over the traditional JSON conversion is the fact, that you can convert object with functions!

If you read this text, you probably don't need any other arguments to start using this plugin. So, go ahead.

And you know what? it's only 0.34K (yes, three hundred and forty bytes :)
