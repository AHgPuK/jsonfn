Overview

JSONfn is a javascript plugin to stringify | parse | clone a javascript object.

With this plugin you can easily serialize any javascript object, save this string in a local storage or on a server and then get the string and convert it back to object.

Plugin is developed to work in both Browser and Node.js environment.

JSONfn Advantage:
Property Type JSON JSONfn Comments
Basic YES YES
Function NO YES Regular JSON ignores functions.
RegExp NO YES Regular JSON returns empty object.
Date NO YES Regular JSON returns a "String" instead of "Date" object.
